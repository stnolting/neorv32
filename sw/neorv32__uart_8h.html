◆ NEORV32_UART_CTRL_enum

enum NEORV32_UART_CTRL_enum

UART control register bits

Enumerator
UART_CTRL_EN 

UART control register(0) (r/w): UART global enable

UART_CTRL_SIM_MODE 

UART control register(1) (r/w): Simulation output override enable

UART_CTRL_HWFC_EN 

UART control register(2) (r/w): Enable RTS/CTS hardware flow-control

UART_CTRL_PRSC_LSB 

UART control register(3) (r/w): clock prescaler select, bit 0 (LSB)

UART_CTRL_PRSC_MSB 

UART control register(5) (r/w): clock prescaler select, bit 2 (MSB)

UART_CTRL_BAUD_LSB 

UART control register(6) (r/w): BAUD rate divisor, bit 0 (LSB)

UART_CTRL_BAUD_MSB 

UART control register(15) (r/w): BAUD rate divisor, bit 9 (MSB)

UART_CTRL_RX_NEMPTY 

UART control register(16) (r/-): RX FIFO not empty

UART_CTRL_RX_FULL 

UART control register(17) (r/-): RX FIFO full

UART_CTRL_TX_EMPTY 

UART control register(18) (r/-): TX FIFO empty

UART_CTRL_TX_NFULL 

UART control register(19) (r/-): TX FIFO not full

UART_CTRL_IRQ_RX_NEMPTY 

UART control register(20) (r/w): Fire IRQ if RX FIFO not empty

UART_CTRL_IRQ_RX_FULL 

UART control register(21) (r/w): Fire IRQ if RX FIFO full

UART_CTRL_IRQ_TX_EMPTY 

UART control register(22) (r/w): Fire IRQ if TX FIFO empty

UART_CTRL_IRQ_TX_NFULL 

UART control register(23) (r/w): Fire IRQ if TX FIFO not full

UART_CTRL_RX_OVER 

UART control register(30) (r/-): RX FIFO overflow

UART_CTRL_TX_BUSY 

UART control register(31) (r/-): Transmitter busy or TX FIFO not empty
