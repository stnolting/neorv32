TRNG control register bits

Enumerator
TRNG_CTRL_EN 

TRNG data register(0) (r/w): TRNG enable

TRNG_CTRL_FIFO_CLR 

TRNG data register(1) (-/w): Clear data FIFO (auto clears)

TRNG_CTRL_SIM_MODE 

TRNG data register(2) (r/-): PRNG mode (simulation mode)

TRNG_CTRL_AVAIL 

TRNG data register(3) (r/-): Random data available

TRNG_CTRL_FIFO_LSB 

TRNG data register(4) (r/-): log2(FIFO size), LSB

TRNG_CTRL_FIFO_MSB 

TRNG data register(7) (r/-): log2(FIFO size), MSB

TRNG_CTRL_NBIT_LSB 

TRNG data register(8) (r/-): log2(number of raw bits) processed for one output byte, LSB

TRNG_CTRL_NBIT_MSB 

TRNG data register(11) (r/-): log2(number of raw bits) processed for one output byte, MSB

TRNG_CTRL_NRO_LSB 

TRNG data register(12) (r/-): number of ring-oscillators, LSB

TRNG_CTRL_NRO_MSB 

TRNG data register(19) (r/-): number of ring-oscillators, MSB

TRNG_CTRL_NINV_LSB 

TRNG data register(20) (r/-): number of inverters in first ring-oscillator, LSB

TRNG_CTRL_NINV_MSB 

TRNG data register(31) (r/-): number of inverters in first ring-oscillator, MSB

int neorv32_trng_check_sim_mode ( void )

Check if TRNG is implemented using SIMULATION mode.

Warning
In simulation mode the physical entropy source is replaced by a PRNG (LFSR) with very bad random quality.
Returns
Simulation mode active when not zero.
